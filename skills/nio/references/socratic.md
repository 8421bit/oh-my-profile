# 苏格拉底式提问

苏格拉底式提问不是为了证明对方错误，而是通过提问帮助对方自己发现答案。

## 六类核心问题

### 1. 澄清类 - 理解对方的意思

- "你说的 X 具体指什么？"
- "能举个例子吗？"
- "你能换个方式表达吗？"

### 2. 假设类 - 检验预设前提

- "你这样想的前提是什么？"
- "为什么你认为这是真的？"
- "如果不是这样呢？"

### 3. 证据类 - 寻找支撑

- "你是怎么知道的？"
- "有什么证据支持这一点？"
- "这个结论是从哪来的？"

### 4. 视角类 - 换位思考

- "从另一个角度看呢？"
- "如果是 XX 会怎么想？"
- "有没有不同的看法？"

### 5. 推论类 - 探索后果

- "如果这样做，会发生什么？"
- "这会导致什么结果？"
- "这和 XX 有什么关系？"

### 6. 元认知类 - 反思提问本身

- "为什么这个问题重要？"
- "我们真正想解决的是什么？"
- "这个问题背后的问题是什么？"

## 使用节奏

```
浅 → 深 → 更深 → 收敛

1. 先用澄清类问题确保理解
2. 用假设类问题挑战前提
3. 用证据类和视角类深入探索
4. 用推论类和元认知类引导收敛
```

## 禁忌

- ❌ 不要连续轰炸问题
- ❌ 不要让对方感觉在"被审问"
- ❌ 每个问题之间给充分思考时间
- ❌ 不要问已经知道答案的问题来"引导"对方
